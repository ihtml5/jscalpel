Usage

Parameter Description
parameter type default value use isRequired
target string/object {} target true
deep boolean false whether or not to copy the target object in depth false
keys string/array/func '' need to access a series of paths, can be a single string or path array false
prefix string undefined public prefix, automatically added for the keys false
callback function function () {} The function that was called when the analysis was successful true
