安装

Es6

npm i jscalpel --save

import jscalpel from 'jscalpel'

Es5

1.下载jscalpel.min.js

2. 在srcipt标签中直接引入

使用

参数说明
参数 类型 默认值 用途 是否必须
target string/object {} 目标对象 是
deep boolean false 是否深度拷贝目标对象 否
keys string/array '' 需要访问的一系列路径，可以是单个字符串或者路径数组 否
prefix string undefined 公共前缀，自动为keys加前缀 否
dynamicKeys string/object undefined 常用在react, angular,vue中根据状态的变化生成不同的keys 否
callback function function () {} 分析成功时调用的函数 是
